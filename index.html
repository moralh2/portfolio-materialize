Trivia Game (JS Time Events)

This game was created to learn how to use JS Time Events -- specifically setInterval in this case. In the trivia game, the timer determines how long to await for the user to respond; or if the user clicks on an answer, an event stops the timer. The timer object was also used to display certain messages for a set number of seconds.

This assignment used Node.js to run different packages to create a command-line interface (CLI) application, called LIRI.
